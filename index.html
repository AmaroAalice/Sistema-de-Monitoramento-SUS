Monitoramento SUS

Plataforma de fluxos de atendimento e agendamento remoto para unidades de saúde.

Sobre o projeto

Este projeto simula uma plataforma de monitoramento de lotação e agendamento para o SUS. Ele possui backend em Python/Flask, banco SQLite e frontend em HTML/CSS/JS.

Ver Dashboard do Paciente Ver Painel Administrativo Instruções de execução
Atenção: o backend Python não é executado diretamente pelo GitHub Pages. Para usar o sistema completo, é necessário executar o servidor localmente conforme as instruções em README.md.

Funcionalidades

  • Listar unidades de saúde e status de lotação.
  • Buscar horários disponíveis de médicos.
  • Efetuar agendamentos remotos.
  • Sugerir a unidade mais próxima com base em geolocalização fictícia.
  • Atualizar manualmente o status de lotação no painel administrativo.
